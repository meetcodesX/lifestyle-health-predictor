# lifestyle-health-predictor
# 🧠 Health Risk Prediction using Machine Learning
## 📌 Project Overview
This project focuses on predicting whether a person is **healthy or at risk** using supervised machine learning techniques.

## 📊 Dataset
* Around 9,800 records
* Includes features like:
  * BMI, Blood Pressure, Glucose Level
  * Sleep, Exercise, Stress Level
  * Smoking, Alcohol, Medical History

## ⚙️ What I Did
* Data Cleaning & Preprocessing
* Handled missing values
* Encoded categorical variables
* Trained ML models

## 🤖 Models Used
* Decision Tree
* Random Forest

## 📈 Results
* Compared model performance
* Selected best model based on accuracy

## 💡 Key Learnings
* Importance of data preprocessing
* Model selection impacts performance
* Real-world datasets are messy

Moving from **Supervised ML → Unsupervised ML**
